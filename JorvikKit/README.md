# JorvikKit

A shared Swift component kit used across Jorvik Software macOS apps.

## Components

| File | Description |
|------|-------------|
| `JorvikAboutView.swift` | Standard About window view |
| `JorvikMenuBarPill.swift` | Menu bar status pill indicator |
| `JorvikMenuBuilder.swift` | Declarative menu construction helper |
| `JorvikSettingsView.swift` | Standard Settings window view |
| `JorvikShortcutRecorder.swift` | Keyboard shortcut capture and display |
| `JorvikUpdateChecker.swift` | Auto-update checker via GitHub releases |
| `JorvikWindowHelper.swift` | Window management utilities |

## Usage

JorvikKit files are embedded directly into each app project (not as a package dependency). Copy the files you need into your project's `JorvikKit/` directory.
